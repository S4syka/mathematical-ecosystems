# Mathematical Ecosystems

The goal of this project is to show people how visually 
comlpex easily implemented automatas can get.

## Running:

First time initialization:

```
mkdir build
cd build
cmake ..
cd ..
```

compilation:

```
cmake --build build/
```

```
build/bin/mathfest
```
